# Исследовательско-разработческий практикум в машинном обучении (ML R&D)

## О курсе

Этот курс предназначен для начинающих ML-инженеров и инженеров-исследователей, а также для тех, кто хочет систематизировать и освежить знания по ML R&D.

В центре курса — практический взгляд на то, **как в индустрии устроена исследовательско-разработческая работа в машинном обучении**:  
как ставятся гипотезы, проектируются эксперименты, сравниваются решения, анализируются ошибки, проверяется устойчивость подхода и оформляется результат так, чтобы его можно было воспроизвести и передать дальше в инженерную работу.

Иными словами, курс о том, **как делать ML-эксперименты так, чтобы их результату можно было доверять**.

---

## Чему посвящён курс

В рамках курса разбирается, как:

- корректно формулировать ML-задачу и гипотезу;
- выбирать метрики и строить честный протокол оценки;
- не обманывать себя случайными улучшениями;
- использовать baseline как опорную систему;
- воспроизводимо запускать и сравнивать эксперименты;
- анализировать ошибки и ограничения модели;
- оформлять результат в виде инженерного и исследовательского артефакта.

---

## Что получит студент по итогам курса

К концу курса у студента будет готовый артефакт уровня портфолио:

- оформленный research report;
- краткая итоговая презентация;
- понятный пайплайн запуска и результаты экспериментов.

## Как сдаются работы

Домашние задания сдаются через **git-репозиторий**.

В репозитории должны быть:

- исходный код;
- README с инструкцией по запуску;
- конфиги или параметры запуска;
- описание экспериментов;
- результаты и выводы.

Репозиторий должен быть оформлен так, чтобы проверяющий мог:

1. понять, что именно делал студент;
2. воспроизвести основной результат;
3. увидеть структуру экспериментов и сделанные выводы.

---

## Общие требования ко всем домашним заданиям

Каждое домашнее задание оценивается не только по качеству результата, но и по качеству исследовательского и инженерного подхода.

- корректная постановка задачи;
- честный протокол оценки;
- наличие baseline;
- воспроизводимость;
- аккуратное описание экспериментов;
- выводы по результатам;
- понятная структура репозитория.
