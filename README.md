# Исследовательско-разработческий практикум в машинном обучении (ML R&D)

## О курсе

Курс создан для того, чтобы дать начинающим инженерам-исследователям (а также тем, кто хочет освежить знания) базовые навыки ведения ML R&D деятельности.  
Кракто - в курсе рассматривается **как в индустрии ставят гипотезы, проектируют эксперименты, честно сравнивают решения, анализируют ошибки и доводят результат до воспроизводимого инженерного артефакта, а также принимают решения о его внедрении**.

Также рассматривается как работать с ML-задачами так, чтобы результату можно было доверять:
- корректно выбирать метрики и протокол оценки;
- не обманывать себя случайными улучшениями;
- строить baseline как опорную систему;
- воспроизводимо запускать эксперименты;
- оформлять результат так, чтобы его можно было передать дальше в инженерную работу. :contentReference[oaicite:1]{index=1}

Формат курса: **практикум / семинары + проект**.  
Длительность: **8 занятий**.  
Автор курса: **Бойматов Юсуфджон Шухратович**, Руководитель группы ML Департамента коммуникаций в RWB, tg: **CuriousOwl**. :contentReference[oaicite:2]{index=2}

### Общие правила сдачи

Домашние задания сдаются через **git-репозиторий**.  
В репозитории должны быть:
- исходный код;
- README с инструкцией по запуску;
- конфиги / параметры запуска;
- описание экспериментов;
- результаты и выводы.

Репозиторий должен быть оформлен так, чтобы проверяющий мог:
1. понять, что именно делал студент;
2. воспроизвести основной результат;
3. увидеть, как устроены эксперименты и выводы.

### Штраф за просрочку

При сдаче после дедлайна максимальный балл за домашнее задание снижается до **70% от исходного**.  
В отдельныъ случаях дедлайн может быть сдвинут по согласованию. :contentReference[oaicite:3]{index=3}

### Требования к домашним заданиям

Каждое домашнее задание оценивается не только по качеству результата, но и по качеству исследовательского и инженерного подхода.

#### Что важно в любом ДЗ
- корректная постановка задачи;
- честный протокол оценки;
- наличие baseline;
- воспроизводимость;
- аккуратное описание экспериментов;
- выводы по результатам;
- понятная структура репозитория.
